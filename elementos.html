Elementos de un sistema de información computacional

1.1 Equipo

Equipo computacional: hardware necesario para que el sistema de información funcione, integrado por computadoras y equipo periférico conectado a ella.

1.2 Recurso humano

El recurso humano son las personas que interactúan con el sistema de información, suministran datos o utilizan el sistema para generar datos.

1.3 Fuente

Son los datos e información que se introducen en el sistema, es decir, son las entradas requeridas para que el sistema informático funcione.

1.4 Programas

Software ejecutado por la computadora, el cual procesa la información de entrada para generar los resultados esperados.

1.5 Telecomunicaciones

Abarca el software y el hardware que permite la transmisión de forma electrónica de texto e imágenes, entre otros.

1.6 Políticas y reglas de operación

Aplicadas a los procedimientos y mecanismos para hacer que una aplicación de computadora trabaje de acuerdo con estas o regir la parte funcional del proceso administrativo de la empresa.
